Well, that's great, because that's exactly what the method of moments does. I mean, take any distribution, say $$X \sim Gamma(\alpha, \beta)$$. We don't know much about this distribution, and we don't really know what is the role of these two parameters, but we can compute / look up the fact that its first two moments, in terms of  are


$$E[X] = \frac{\alpha}{\beta}$$

$$Var[X] = \frac{\alpha}{\beta^2}$$


Ok. So the method of moments says, that, if you want to use your data to backsolve for the value of these parameters, you should equate the first moment expressions above to the expression for the first moments using your data. That is, whenever we get data $$X_1, \cdots, X_n$$ from this distribution, we just equate


$$\frac{\alpha}{\beta} = \frac{X_1 + \cdots + X_n}{n}$$ 

$$\frac{\alpha}{\beta^2} = \frac{1}{n}\sum_i \left(X_i - \frac{X_1 + \cdots + X_n}{n} \right)^2$$


And then solve this system of equations for the unknown parameters $$\alpha, \beta$$.
